# 🚀 دليل النشر - TalentHub

## تم إعداد المشروع للنشر المجاني على:
- **Frontend**: Vercel (مجاني)
- **Backend**: Replit (مجاني)

---

## خطوة 1️⃣: نشر الـ Frontend على Vercel

1. اذهب إلى **vercel.com**
2. سجل دخول بـ GitHub
3. اضغط **"New Project"**
4. اختر الريبو **`faresosama200/Elking2`**
5. اضغط **Deploy** ✅

**الرابط**: `https://elking2-[random].vercel.app`

---

## خطوة 2️⃣: نشر الـ Backend على Replit

1. اذهب إلى **replit.com**
2. سجل دخول بـ GitHub
3. اضغط **"Import"** من GitHub
4. اختر الريبو **`faresosama200/Elking2`**
5. اختر **Root: backend**
6. اضغط **"Import"** ✅

**الرابط**: `https://Elking2-[username].replit.dev`

---

## خطوة 3️⃣: تحديث الـ Frontend بـ Backend URL

بعد ما تاخد الرابطين:

```javascript
// في dashbord/api.js و login.html و register.html
const RENDER_API_URL = "https://Elking2-[username].replit.dev/api";
```
